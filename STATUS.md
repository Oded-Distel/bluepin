# איפה אנחנו בעבודה

## מה הפרויקט

**BluePin** – כלי לסימון תכניות בנייה. הלקוח מקבל קובץ HTML יחיד, מסמן וולבוקסים/מצלמות/כבלים על PDF, ומייצא PDF סופי עם דף תכולות.

## הקובץ הפעיל

**`marker-tool.html`** – הקובץ היחיד שעובדים בו.

> ⚠️ `סימון-וולבוקס.html` ו-`marker-tool-standalone.html` הם סנפשוטים ישנים, **לא** גרסת העבודה.

## מה כבר קיים ✓ (עודכן 2026-06-10)

### פיצ'רים בסיסיים
- אייקון Wallbox אדום עם שם
- אייקון מצלמה SVG כחול עם עדשה, אלומת ראייה וסיבוב
- כבל מצלמה ↔ Wallbox
- העלאת PDF/תמונה רקע + בורר דפים

### תכולת Wallbox (הפיצ'ר הגדול, סבב #1 בספק החדש) ✅
- מבנה `contents: [{name, qty}]` + מיגרציה ל-localStorage ישן
- פאנל צד 320px (RTL, ARIA), 9 פריטים דיפולטיים + מותאמים אישית
- combobox עם autocomplete + ניווט מקלדת (חצים/Enter/ESC)
- stepper כמות, מחיקה לפי שורה
- badge על המרקר עם סכום הכמויות + tooltip
- עריכת שם דרך הפאנל; double-click → focus על שם

### UI מאוחרים (אחרי הסיום של תכולת WB)
- **פאנל צד שמאל**: כפתורי `＋ WB`, `＋ מצלמה`, `↔ כבל`, `↶ בטל`, `העלה לוגו`
- **כותרות מקטעים** ברשימה הצדדית: "Wallboxים (N)" אדום, "מצלמות (N)" כחול
- **כרטיסים** דו-טוריים: שם בצד אחד + תכולה בצד שני
- **Undo** עם היסטוריה (עד 50) + Ctrl/Cmd+Z; סוגר גם את פאנל ה-WB
- **Logo upload**: thumb בצד + overlay בפינת התכנית + מודפס ב-PDF
- **גרירת כבל מהעדשה** של המצלמה לרווח ריק → יוצרת WB חדש שם ומחברת כבל
- **3 WB + 3 מצלמות אוטומטיים** כשפותחים PDF חדש (markers ריק)
- **PDF export מלא** (jsPDF, A4): עמוד 1 = תכנית + לוגו, עמוד 2 = רשימת תכולת WBs

### Cleanup / removed
- כפתורי "וולבוקס", "מצלמה", "כבל" בסרגל העליון הוסרו (יש בצד)
- textarea של JSON, כפתורי "ייבא/ייצא JSON" הוסרו (PDF זה היעד)
- מקטע "כבלים" מהרשימה הצדדית הוסר (הכבלים נראים על התכנית)

## מה נשאר מהספק הראשי

1. **חדר (Room)** – מלבן עם שם
2. **ייצוא PDF** – חלקית מומש (קיים PDF בסיסי + עמוד תכולות), חסר: לוגו של החברה כותרת מעוצבת, מספרי דפים, תאריך
3. **ניהול פרויקטים מרובים** – יצירה/שמירה/רשימה/פתיחה

## דיון פתוח (לשיחה הבאה)

הלקוח רוצה שתי תוספות הקשורות לאריזה:

### א. שליטה בגישה ללקוח
שתי גישות נשקלות:
- **מהיר ולא-באמת-בטוח (15-30 דק' לי, 0 לך)**: hash סיסמה משובץ ב-HTML עם פרומפט בטעינה. מי שמכיר View-Source עוקף.
- **אמיתי עם ארוח (1-2 שעות סך)**: Cloudflare Pages + Zero Trust Access. שלי: 20 דק' (הקמת ריפו). שלו: 30-60 דק' (חשבון CF, חיבור GitHub, מדיניות).

המשתמש עדיין לא בחר כיוון. ההמלצה שלי: לקוח אחד נחמד → המהיר; תשתית עתידית → ארוח עכשיו.

### ב. גרסאות סרטוטים (פרויקטים מרובים)
"פרויקטים" בעלי שם ב-localStorage (slot per project), עם UI של dropdown + כפתורי חדש/שכפל/מחק/שנה-שם + ייצוא/ייבוא JSON לגיבוי. מגבלות: per-browser, ~5MB.
